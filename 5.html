Las listas HTML permiten a los desarrolladores web agrupar un conjunto de elementos relacionados en las listas.
Listas Desordenadas
Una lista desordenada comienza con el UL etiqueta. Cada elemento de lista comienza con el LI etiqueta. Los elementos de la lista se marcarán con viñetas ( círculos negros pequeños ) de forma predeterminada.
Listas Ordenadas
Una lista ordenada comienza con el OL etiqueta. Cada elemento de lista comienza con el LI etiqueta. Los elementos de la lista se marcarán con números de forma predeterminada.

Listas de Descripción
HTML también admite listas de descripciones. Una lista de descripción es una lista de términos, con una descripción de cada término. los DL la etiqueta define la lista de descripción, el DT la etiqueta define el término ( nombre ), y el DD la etiqueta describe cada término.
